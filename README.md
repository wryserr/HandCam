# HandCam
*HandCam is a Streamer/Content Creator focused handheld camera script for Orion Drift with lots of customization.*

### Features:
- Selfie Mode with Selfie Stick & Camera Flip
- Camera Passing, place the camera on a players head to give them temporary ownership
- Full Customization of controls, you are able to change what actions different inputs will cause. For example: "If Inside Head while flipped then Third Person"
- Camera Settings like FOV, Recall Settings, etc.
- Smoothing and Mode-Specific settings for each mode
- First Person, Third Person and Follow Player modes
- Recalling and Auto Recall, camera will automatically recall if you drop it and go too far away

### Installation:
Unzip the .zip file and put the folder into "Documents\Another-Axiom\A2\Cameras\Behaviors"
